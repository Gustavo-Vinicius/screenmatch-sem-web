# 📺 ScreenMatch

Aplicação Java desenvolvida durante o curso **Java: trabalhando com lambdas, streams e Spring Framework** da Alura.

O projeto consome dados da API OMDb para buscar informações sobre séries, temporadas e episódios, utilizando conceitos modernos do ecossistema Java como:

- Streams API
- Expressões Lambda
- Programação Funcional
- Consumo de APIs REST
- Manipulação de coleções
- Orientação a Objetos
- Spring Boot

---

# 🚀 Tecnologias Utilizadas

- Java 17+
- Spring Boot
- Maven
- Streams API
- Lambda Expressions
- API REST
- OMDb API

---
